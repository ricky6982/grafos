Creación del Mapa de Recorridos

Representación de Senderos, Intersecciones y Puntos de Referencia
Mejora en la interfaz de usuario para la creación de trayectos.
Objetivo: El objetivo en esta version del prototipo es implementar un algoritmo de la ruta mas corta entre dos nodos.
Se agregaron las siguientes funciones al prototipo:
  • El usuario puede establecer el valor de la distancia que existe entre dos nodos, para ello selecciona un arco del grafo y luego establece el valor de la distancia.
  • El usuario puede cambiar el tipo de nodo a Intersección o Punto de Referencia, de forma predeterminada cuando un nodo es creado tiene el valor de Intersección.
  • Se agrego libreria para calcular el camino más corto mediante el algoritmo de Dijkstra.
